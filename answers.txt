• What is a gitignore file and what should it include for a Java project?
	The gitignore file in git includes the files or patterns that are to be ignored during a commit. 
	A git ignore file for a java project inlcudes class files, log files, BLueJ files, and package files

• What does fast-forward in Git mean?
	When we merge a branch which has no other user working or modifying that branch, git can simply merge it by moving the pointer forward. This merging is called fast forward.

• Why should you have different branches?
	Having different branches in a repo allows user to implement new code without affecting the old running code. It means that a user can work on a new version of the project and he can merge it or revert back to the original code easily.
